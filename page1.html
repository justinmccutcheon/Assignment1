These are some things we've learned in this class

  1. Semantic Tags
    1. These divide your page into readable chunks
    2. They don't do anything without CSS formatting
  2. CSS sheets
    1. This is a formatting sheet that dictates what your page looks like
    2. You can change all kinds of things like fonts, colors and text alignment
    3. By using the style sheet you can change the whole site with one page
  3. Posting to a git
    1. You can quickly upload to a repo by pushing it though the command line
    2. Your posts can be divided into multiple pushes so things can be easily rolled back
    3. They also provide your own personal web page
