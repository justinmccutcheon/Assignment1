HTML Tags HTML Attributes
  • "p" starts a paragraph
  • "ul" makes an unordered list
  • "ol makes an ordered list
  • "li" determines elements in that list
  • "img" is the tag for an image
  1. "src" defines where an image is coming from
  2. "type" in an ordered list changes the bullet type
  3. "width" can change the size of a table
  4. "href" determines where a link points to
  5. "style" can be used to change elements of a paragraph
